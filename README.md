# EchartsDemo

一套基于 [Ng-zorro-antd](https://github.com/NG-ZORRO/ng-zorro-antd)【ANT DESIGN】 和     [ng-alain](https://github.com/cipchk/ng-alain)的企业后台模板。

## 运行

### 直接 clone git 仓库

```bash
$ git clone --depth=1 https://github.com/SevenLonelyDay/echartsdemo my-project

cd my-project

# 安装依赖包
npm install

# 启动
npm start

# 使用HMR启动
npm run serve:hmr
```
